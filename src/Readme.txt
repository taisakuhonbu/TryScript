■■■■
TryScript
■■■■

ソースコードの解説を見るため、それを作るためのプログラムです。

◆◆◆◆
操作マニュアル
◆◆◆◆

〇〇〇〇
学習者向け
〇〇〇〇

画面に最初のソースコードの概要が表示されます。
上部少ししたの「後」ボタンを押すとソースコードを表示しながらの解説に切り替わり、注目すべき部分ががハイライトされます。
「前」ボタンはひとつ前の解説に戻ります。
そのソースコードの解説が最後まで行ったら、下部にある「次」ボタンを押してください。次のソースコードの解説が始まります。
「戻」ボタンはひとつ前のソースコードに戻ります。
「戻」と「次」の間にあるドロップダウンリストでソースコードを直接指定して表示できます。

上部にある「文字サイズ」の左にある５つのラジオボタンで文字サイズを変更できます。

上部にある「選択」ボタンを押すとファイル選択ダイアログが表示されます。FileList.txt を選択することで、ほかのソースコードセットの学習ができます。
起動直後は、実行中のフォルダーにある FileList.txt を読み込みます。

〇〇〇〇
教育者向け
〇〇〇〇

画面右上に 40px に見えない正方形があります。ここで右クリックするとファイル選択ダイアログが表示されます。解説したいソースコードを選択すると、FileList.txt と解説の XML ファイルが作られます。複数選択が可能で、ソースコードファイル名に .xml の拡張子が追加されます。見えない正方形をダブルクリックすると、編集モードに切り替わります。

下部のドロップダウンリストに表示されているソースコードが解説作成対象です。「後」「前」「次」「戻」ボタンは常に有効で、表示されている解説文の更新、または前後に追加ができます。

編集は、まずハイライトしたい文字列を指定します。編集エリア上部のテキストボックスに文字列または正規表現を入力して、単純文字列または正規表現をラジオボタンで選択、スキップ数を指定します。確認ボタンを押すことで文字列がハイライトされます。文字列でなく複数行をハイライトする場合は、行指定と行数で行番号を入力してください。なお、ハイライトなしにする場合は、単純文字列または正規表現で空文字列を指定するか行指定で 0 を入力します。解説文は大きなテキストボックスに HTML のソースコードで入力します。&amp; など、お忘れなく。

「前に追加」「更新」「後に追加」のいずれかのボタンを押すと保存されます。削除はできません。XML ファイルをテキストエディターで開いて、desc 要素を削除してください。
